LANGKAH 1: JALANKAN SERVER
---------------------------

Windows:
  1. Double-click file: start-test.bat
  
Mac/Linux:
  1. Buka terminal
  2. Ketik: python3 -m http.server 8000
  3. Tekan Enter


LANGKAH 2: BUKA BROWSER
------------------------

Buka browser (Chrome/Firefox/Edge) dan ketik di address bar:

  http://localhost:8000/test-ultra-simple.html


LANGKAH 3: LIHAT HASILNYA
--------------------------

Jika SUKSES, Anda akan melihat:
  ‚úì 3 objek 3D berputar (box merah, sphere hijau, box biru)
  ‚úì Console log hijau di bawah layar
  ‚úì Pesan "SUKSES! KALYTHESAINZ BERJALAN DENGAN SEMPURNA!"


TROUBLESHOOTING
---------------

Masalah: Layar putih kosong
Solusi: Pastikan URL dimulai dengan http:// (bukan file://)

Masalah: Error di console
Solusi: Tekan F12, screenshot error, tanyakan ke developer

Masalah: Server tidak jalan
Solusi: Install Python dari https://www.python.org/downloads/
